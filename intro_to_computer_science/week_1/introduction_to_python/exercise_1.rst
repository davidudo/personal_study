Week 1 (Exercise 1)
===================

1. What is the difference between an Algorithm and a Program?

   Answer
   ------
   An algorithm is a conceptual idea, a program is a concrete instantiation of
   an algorithm.

2. True or False? A computational mode of thinking means that everything can be
   viewed as a math problem involving numbers and formulas.

   Answer
   ------
   True

3. True or False? Computer Science is the study of how to build efficient
   machines that run programs.

   Answer
   ------
   False

4. The two things every computer can do are:

   Answer
   ------
   1. Perform calculations
   2. Remember results
